## How to Use

Each prompt lives in its category folder as a markdown file with:
- **Use case** — what problem it solves
- **Prompt** — the exact text to use
- **Notes** — tips, model recommendations, variations

To add a prompt, say: `"add this prompt: [your prompt]"` — it gets categorized and committed automatically.
